# Lists

Lists are used to store multiple arguments in 1 variable, in the format of an array.

#### **`main.???`**
```
int main(argc, argv)
{
   var littleList = [1, "salut", "hey"];

   print(littleList);
}
```

To learn how to compile the file, [check this out!](./compile_and_run.md)

```
> ./GLaDOS -r executable
[1, "salut", "hey"]
```
