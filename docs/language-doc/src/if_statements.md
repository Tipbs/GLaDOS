# If Statements

Usage of if statements is pretty straight forward and easy to use.

```
if (condition) {
	...
} else {
	...
}
```

If you'd like to see a real example

#### **`main.???`**
```
fn main(argc, argv)
{
	if (4 > 9) {
		print("It is true!")
	} else {
		print("It is false!")
	}
}
```
